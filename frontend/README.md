# GEI Adeptify Frontend

Frontend de la plataforma educativa GEI Adeptify construido con Next.js 14.

## Tecnologías

- Next.js 14 con App Router
- React 18
- TypeScript
- Tailwind CSS
- NextUI
- Framer Motion

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Producción

```bash
npm start
```
